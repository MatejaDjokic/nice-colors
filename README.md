## Features

Introducing NiceColors - your go-to solution for adding a touch of elegance to your Flutter projects!

## Getting started

In your flutter project add the dependency:

```yaml
dependencies:
  nice_colors: ^1.1.2
```

Import the package:

```dart
import 'package:nice_colors/nice_colors.dart';
```
